# 🌟 Overview

**HandWave** is an AI-powered hand gesture recognition system built using **Python**, **OpenCV**, and **MediaPipe**.

The project allows users to interact with their computer completely touch-free using hand gestures through a webcam.

Using real-time hand landmark detection, the system can:

- 🎵 Control System Volume
- 💡 Adjust Screen Brightness
- 🖱️ Move the Mouse Cursor
- 👆 Perform Mouse Clicks
- 📸 Take Screenshots
- ⏯️ Control Media Playback
- 🔇 Mute / Unmute Audio

---

# 🚀 Features

| Feature | Description |
|---------|-------------|
| 🖱️ Virtual Mouse | Move cursor using hand movement |
| 👆 Gesture Click | Click using pinch gesture |
| 🎵 Volume Control | Adjust volume using finger distance |
| 💡 Brightness Control | Control brightness with gestures |
| 📸 Screenshot Capture | Take screenshots using "L" gesture |
| 🔇 Mute / Unmute | Fist gesture toggles mute |
| ⏯️ Media Controls | Swipe gestures for next/previous track |
| 🎥 Real-Time Detection | Smooth hand tracking using webcam |
| 📊 OSD Interface | On-screen volume & brightness indicators |

# 🖐️ Gesture Controls

| Gesture | Action |
|---------|--------|
| ✋ Hand Movement | Move Mouse Cursor |
| 🤏 Pinch Gesture | Left Click |
| 👌 Thumb + Index Distance | Volume / Brightness Control |
| ✊ Fist Gesture | Mute / Unmute |
| 📐 "L" Gesture | Take Screenshot |
| 👉 Swipe Right | Next Track |
| 👈 Swipe Left | Previous Track |
